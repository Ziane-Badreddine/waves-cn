## Features

- **Waveform visualization** ŌĆö renders a decoded waveform on a Canvas via wavesurfer.js; click or drag anywhere on it to seek.
- **Seek bar** ŌĆö a shadcn `Slider` below the waveform gives precise scrubbing independently of the waveform.
- **Volume & mute** ŌĆö slider-based volume control with a mute toggle that preserves the last set level.
- **Restart** ŌĆö jumps back to 0 and resumes playback instantly via `ws.setTime(0)`.
- **Loading state** ŌĆö a spinner overlays the waveform while the audio is being fetched and decoded.
- **Shadcn theming** ŌĆö colors are resolved at runtime from Tailwind utility classes so the component inherits your active theme, including dark mode, without any extra configuration.

### With event callbacks

Because `onPlay`, `onPause`, `onFinish`, and `onTimeUpdate` are functions, they must be passed from a **Client Component**.

```tsx
"use client"

import { AudioPlayer } from "@/components/ui/audio-player"

export function MyPlayer() {
  return (
    <AudioPlayer
      src="/audio/track.mp3"
      onPlay={()  => console.log("Playing")}
      onPause={() => console.log("Paused")}
      onFinish={() => console.log("Finished")}
      onTimeUpdate={(current, total) =>
        console.log(`${current.toFixed(1)}s / ${total.toFixed(1)}s`)
      }
    />
  )
}
```

## API Reference

### AudioPlayer

| Prop | Type | Default | Description |
|---|---|---|---|
| `src` | `string` | ŌĆö | **(Required)** Audio URL. Changing it destroys and reinitializes WaveSurfer. |
| `title` | `string` | ŌĆö | Track title displayed above the waveform. |
| `defaultVolume` | `number` | `0.8` | Initial volume `0`ŌĆō`1`. |
| `autoPlay` | `boolean` | `false` | Start playing as soon as audio is ready. |
| `onPlay` | `() => void` | ŌĆö | Fired when playback starts. |
| `onPause` | `() => void` | ŌĆö | Fired when playback pauses. |
| `onFinish` | `() => void` | ŌĆö | Fired when the track ends. |
| `onTimeUpdate` | `(currentTime: number, duration: number) => void` | ŌĆö | Fired each tick with current time and total duration. |
| `waveHeight` | `number` | `64` | Waveform canvas height in px. |
| `barWidth` | `number` | `3` | Waveform bar width in px. |
| `barGap` | `number` | `2` | Gap between bars in px. |
| `barRadius` | `number` | `2` | Border radius of each bar. |
| `minPxPerSec` | `number` | `1` | Minimum pixels per second (zoom level). |
| `className` | `string` | ŌĆö | Extra classes on the outer `<Card>`. |
